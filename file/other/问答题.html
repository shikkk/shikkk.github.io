# 1. 输入url页面加载过程

  • DNS解析：根据域名解析IP地址（网址不同区域ip不同）
  • 浏览器会根据ip地址向服务器发起http请求
  • 服务器处理http请求，并返回给浏览器
  • 浏览器拿到（html、css、js）并渲染页面
    • 根据HTML代码生成DOM Tree
    • 根据css代码生成CSSOM
    • 将DOM Tree和CSSOM整合形成Render Tree
    • 根据Render Tree渲染页面
    • 遇到 script 标签则暂停渲染，优先加载并执行js代码，完成再继续
    • 直到把Render Tree渲染完成
