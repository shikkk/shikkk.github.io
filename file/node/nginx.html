# nginx

# 1.安装

  • win：http://nginx.org/en/download.html (opens new window)
  • mac: brew install nginx

# 2.配置文件

  • win：c:\nginx\conf\nginx.conf
  • mac：/usr/local/etc/nginx/nginx.conf

# 3.基本命令

  • 测试配置文件格式是否正确 nginx -t
  • 启动 nginx ： 重启 nginx -s reload
  • 停止nginx -s stop
