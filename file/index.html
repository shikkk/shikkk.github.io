# 前言

你永远赚不到超出你认知范围外的钱
除非靠运气
但是靠运气赚的钱
最后往往又会靠实力亏点这是必然
你所赚的每一毛钱
都是你对这个世界认知的变现
亏的每一毛钱
都是对这个世界认知的缺陷
这个世界最大的公平在于
当一个人的财富大于自己的认知
这个社会就会有百种千种方法收割你
直到你的认知和财富相匹配
