# linux常用命令

命令 备注 例子
ssh xxx@192.168.12.9 登录远程服务器 xxx是用户名
ls 查看当前文件列表(平铺)
ll 查看列表
clear 清屏
mkdir 创建文件夹
rm -rf 删除文件夹加-rf，删除文件不需要
mv 修改文件名 mv index.html index1.htm
mv 移动文件 mv index.html ../src
cp 拷贝文件
touch 新建文件
vi/vim 新建一个文件并且打开编辑 按“i”键开始编辑 esc退出、q退出
cat 查看一个文件
grep 查找一个关键字 grep "babel" package.json
